Chi ha inventato questo linguaggio?

L'inventore del linguaggio html è Tim Berners-Lee nei primissimi anni novanta.


Quando è stato inventato?

Tim Berners-Lee ha inventato il linguaggio html negli anni novanta. Esattamente nel '89.


Quali sono gli scopi di questo linguaggio?

Gli scopi dell'html sono sicuramente creare siti, integrare i media,collaborare con altri siti per diversi tipi di stili,e facilitare l'uso per le persone con disabilità.


Che cos'è il WEB?

Il web, o world wide web, è un sistema oppure una grande piattaforma dove tutti possono navigare, collegati tra loro via internet tramite browser.


Che problemi risolveva il WEB ?

Il web risolveva problemi di accesso rapido, permettendo la condivisione e consultazione di documenti ipertestuali su scala globale. Ha reso più semplice collegare, cercare, diffondere, praticamente ci ha semplicato la vita.


Che differenza c'è tra web, html ed internet?

Il web è un sistema di documenti e risorse varie interconnesse. L'html è un linguaggio di markup usato per creare le pagine web, e infine internet è l'infrastruttura globale di reti interconnesse che consente la trasmissione di dati vari liberamente.
